A React-based online flower shop where users can browse products, search, add items to the cart, and complete checkout.

## Description

This project was created as part of the **JavaScript Frameworks Course Assignment**.  
**Bloom & Co.** is an online flower shop built with React, featuring product browsing, live search, reusable components, cart management, a full checkout flow, and a validated contact form.

Key functionality includes:

- Browse products from the Noroff Online Shop API  
- Real-time search filtering  
- Product detail pages with reviews and discounts  
- Shopping cart with live updates  
- Full checkout process  
- Contact form with validation  
- Responsive design for mobile and desktop  

## Built With

- React 19  
- React Router 6  
- Tailwind CSS  
- Vite  
- Noroff Online Shop API v2
